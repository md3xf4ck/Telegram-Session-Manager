# Telegram Session Manager

Многоаккаунтный менеджер сессий Telegram для автоматической защиты аккаунтов и централизованного получения кодов авторизации.

## Описание

Этот проект предназначен для анализа сессий Telegram с нескольких аккаунтов. Если у пользователя есть несколько аккаунтов, через этот софт он сможет автоматически завершать все новые сессии и получать коды авторизации от Telegram.

## Преимущества

- **Автоматическая защита аккаунтов** - все новые сессии автоматически завершаются
- **Централизованный контроль** - все коды авторизации со всех аккаунтов поступают в одну консоль
- **Удобство для владельцев нескольких аккаунтов** - не нужно переключаться между аккаунтами для получения кодов

## Возможности

- Подключение нескольких Telegram аккаунтов
- Автоматическое завершение новых сессий
- Получение кодов авторизации в единую консоль
- Мониторинг активности на всех подключенных аккаунтах

## Установка

1. Склонируйте репозиторий
2. Установите зависимости из requirements.txt
3. Настройте конфигурацию под свои аккаунты

## Использование

Запустите основной скрипт:

```bash
python main.py
```

## Требования

- Python 3.8+
- Библиотеки: см. requirements.txt
